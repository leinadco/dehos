Acest proiect este o galerie de imagini interactivă, care permite utilizatorilor să vizualizeze o colecție de fotografii organizate pe categorii. Folosind HTML, CSS și JavaScript, proiectul include următoarele funcționalități:

Structura HTML: Pagina este organizată cu un meniu de navigare, secțiuni pentru imagini și butoane de navigare. Imaginile sunt prezentate într-un container și sunt grupate pe categorii (ex: „moon”, „sunset”, „nature” etc.), iar utilizatorii pot filtra galeria pe baza acestor categorii.

Design Responsiv: Folosind media queries în CSS, designul se adaptează pentru diferite dimensiuni de ecran, asigurând o experiență vizuală plăcută pe dispozitivele mobile și desktop.

Temă de zi/noapte: În funcție de ora zilei, aplicația schimbă tema, trecând automat de la o temă de zi la una de noapte (background întunecat, text alb) pentru a îmbunătăți vizibilitatea și confortul utilizatorului.

Navigare și filtrare: Utilizatorii pot naviga prin imagini, iar pentru fiecare categorie există opțiunea de a filtra și vizualiza doar imaginile corespunzătoare acelei categorii. Există butoane pentru a vizualiza mai multe sau mai puține imagini, în funcție de preferințele utilizatorului.

Funcționalități suplimentare: Fiecare imagine are o etichetă de categorie, iar la clic pe o imagine se afișează URL-ul acelei imagini în consolă.

Această aplicație este un exemplu de utilizare a tehnologiilor web pentru a crea o galerie de imagini dinamică, cu funcționalități interactive și design responsive.
